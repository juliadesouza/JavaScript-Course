1) Dot vs Bracket Notation
    aluno.nome; // dot notation
    aluno['nome']; // bracket notation
    OBS: in bracket notation, the advantage is that we can pass 
    an expression. Ex:
        const key = 'nome'
        aluno['primeiro'+key];
        aluno['sobre'+key];
    
    To add a new attribute, do:
    - nome.novoAtributo or nome['nomeAtributo'];

2) Object Method 
    A) we can add functions to objects. It has to be a expression. 
        const julia = {
            name: "julia",
            job: "software developer",
            birthYear: 2000,
            calcAge: function(birthYear){
                return 2022-birthYear;
            }
        }

        julia.calcAge(2000);
        julia['calcAge'](2000);

    B) To use the birthYear defined inside the object, we will use the keyword THIS.
        const julia = {
            name: "julia",
            job: "software developer",
            birthYear: 2000,
            calcAge: function(){
                return 2022-this.birthYear;
            }
        }
        julia.calcAge();

    C) We can store the function return in a property of the object
        const julia = {
                name: "julia",
                job: "software developer",
                birthYear: 2003,
                calcAge: function(){
                    this.age = 2022-this.birthYear;// create a new property;
                    return this.age;
                }
            }
            console.log(julia.calcAge());
            console.log(julia.age);
   

 
ITERATION
1) FOR
    for(let rep=1; rep<=10; rep++){
        console.log(`Lifting weights repetition ${rep}`);
    }
2) Looping Arrays, Breaking and Continuing
    A) Looping Arrays
        const names = ["Julia", "Hanna", "Ana Laura", "Elisangela", "Marcio"];
        for(let i=0; i<names.length; i++){
            console.log(names[i]);
        }

    B) Continue
        const names = ["Julia", "Hanna", "Ana Laura", "Elisangela", "Marcio"];
        for (let i = 0; i < names.length; i++) {
            if (i % 2 == 0) continue; // will not execute what it becomes after
            console.log(names[i]);
        }
    C) Break
        const names = ["Julia", "Hanna", "Ana Laura", "Elisangela", "Marcio"];
        for (let i = 0; i < names.length; i++) {
            if (i % 2 != 0) break; // will not execute the loop anymore
            console.log(names[i]);
        }
3) Looping Backwards and Loops in Loops
    A) Backwards
        const names = ["Julia", "Hanna", "Ana Laura", "Elisangela", "Marcio"];
        for (let i = names.length-1; i >= 0; i--) {
            console.log(names[i]);
        }
    B) Loops in Loops

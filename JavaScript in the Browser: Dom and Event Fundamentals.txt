WHAT'S THE DOM 
- Document Object Model: structured representation of HTML 
documents. Allows JavaScript to acess HTML elements ans styles to
manipulate them.  
- Its a connection between html documents and JavaScript code.
- Tree structure, generated by browser on HTML load.
- Document Object: special object point that is the entry point 
to the DOM.
